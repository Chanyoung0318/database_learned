인터넷이란?
전세계 컴퓨터들을 하나로 연결하는 거대 통신망을 의미

웹이란?
World Wide Web, WWW, W3 와 같은 의미로 인터넷을 이용해 문서를 주고 받는 서비스를 의미

웹에서 사용자 간 문서를 주고 받기 위해서 사용되는 약속된 문서가 HTML


웹표준이란 웹을 만들 때 사용하는 약속된 방법을 의미
html5가 곧 웹표준

html (Hyper Text Markup Language)

하이퍼 텍스트 : 책처럼 순서대로 위치한 것이 아닌 클릭하면 그 때 그 때 필요한 내용을 보여주는 문서

마크업 랭귀지 : 특정 글에서 특정 부분의 색과 크기를 글로 표현하고 자 할 때 사용하는 언어를 의미

html 에디터 : html 문서를 쉽게 만들기 위해서 편집하는 응용프로그램을 html 에디터라고 함, ex) vscode

웹브라우져 : html 문서의 내용을 해석해 일반 사용자가 볼 수 있도록 화면에 보여주는 응용 프로그램

<!DOCYTYPE HTML> 부분은 해당 문서가 html5 문서임을 의미
<html> 모든 html문서는 해당 태그 내부에 기술되어야 함
<head> 부분은 부라우저 화면과 관련없는 html문서를 기술함
<title> 해당 문서가 열린 브라우저 탭에 이름으로 사용
<meta charset="UTF-8"> 해당 문서의 인코딩 방식이 utf-8 임을 기술한 것
<body> 브라우저에서 보여주는 화면과 관련된 html 문서를 기술함
